Metodologia & Fontes

  • Fonte — Bitcoin Lab API (researchbitcoin.net), a mesma que alimenta painéis como o yldlab.xyz. Snapshot diário gravado no repo; a página lê dados estáticos (token nunca exposto).
  • MVRV / STH / LTH MVRV — Market Value ÷ Realized Value, total e segmentado por holders de curto e longo prazo. <1 = mercado abaixo do custo base do grupo.
  • Mayer Multiple — preço ÷ média móvel de 200 dias (calculado aqui). <0,8 zona de compra; >2,4 sobreaquecido.
  • Realized Price — custo médio on-chain da rede (preço em que cada moeda se moveu pela última vez).
  • LTH SOPR — lucro/prejuízo realizado pelos holders de longo prazo. <1 = vendendo no prejuízo (capitulação/fundo).
  • AVIV — Active Value to Investor Value (framework Cointime). <0,8 subvalorizado; >2,4 sobrevalorizado.
  • CVDD — Cumulative Value-Days Destroyed: piso de ciclo histórico (derivado dos coinblocks destruídos). Mostrado como múltiplo preço/CVDD.
  • Não é recomendação. Thresholds são heurísticas de ciclo, não gatilhos mecânicos. "BTC Agora" é spot ao vivo (CoinGecko); o resto é fechamento diário (lag ~1 dia).
